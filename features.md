# 🌟 功能概览

Ptmj Desktop 致力于为您提供最优雅的本地资源与网络书签管理体验。摒弃繁杂的表格，我们用现代化的卡片与树形结构重塑了信息流。

---

## 📁 智能资源管理器

将杂乱无章的本地试卷、课件、学习资料进行结构化、可视化的管理。

- **无缝树形视图**：极速加载本地嵌套文件夹，支持一键折叠/展开所有节点，找文件快人一步。
- **文件元数据抽屉**：选中文件即可在右侧平滑呼出详情抽屉，清晰查看文件大小、分类、贡献者、年份等业务信息。
- **全局搜索与过滤**：顶部的胶囊搜索框支持对试题和资料进行实时过滤，无需等待。
- **快捷指令**：支持文件后缀名自由隐藏、一键复制文件名/科目，提升操作效率。

---

## 🔖 书签便当盒 (Bento Box)

不仅是本地文件，您的网课链接、学术论文、优质博客同样需要一个优雅的“家”。

- **沉浸式添加体验**：采用高度类似 Apple 与 Notion 的弹窗设计，表单输入如丝般顺滑。
- **封面自动云同步**：支持本地选择封面图片（实时本地预览），保存时自动直传至 MinIO 云端对象存储，保障数据高可用。
- **三级分类视图**：书签不再是杂乱无章的列表，而是按 `专栏 -> 科目 -> 具体书签` 的层级，以带有精美图标的树形结构优雅呈现。

---

## 🎨 极致的 UI 交互美学

我们相信，工具的美感能极大地提升学习与工作的愉悦感。

- **全屏毛玻璃 (Glassmorphism)**：深度定制的透明模糊效果。无论您上传多么复杂的自定义壁纸，应用内的卡片与文字都能保持极高的对比度与清晰度。
- **自适应深色模式**：无论是刺眼的白昼还是专注的深夜，应用都能完美跟随系统主题，提供最舒适的暗黑质感。
- **微交互动效**：从弹窗的 Apple 弹性阻尼动画、全局丝滑的 Toast 提示，到悬浮按钮（Pill Button）的发光流光效果，每一个细节都经过精心打磨。

---

## ⚡ 性能与架构

- **Electron + Vue 3 引擎**：采用最新的跨平台桌面架构，前端基于 Vite 驱动，保证了媲美原生应用的极速启动与响应。
- **渲染管线优化**：彻底剥离了冗余的滤镜与过渡重绘，针对大列表和复杂树形结构进行了深度的 GPU 硬件加速调优，告别掉帧与卡顿。
- **全栈协同**：完美对接 RuoYi 后端生态、MySQL 数据库与 Redis 缓存，不仅仅是好看的壳子，更是企业级的数据流通管道。
